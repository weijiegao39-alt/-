# 旅行者2.0 iPhone PWA版

## 部署到 GitHub Pages
1. 新建 GitHub 仓库，例如 `traveler2-pwa`。
2. 上传本文件夹内所有文件：`index.html`、`style.css`、`app.js`、`manifest.json`、`service-worker.js`、`icons/`。
3. 打开仓库 Settings → Pages。
4. Source 选择 `Deploy from a branch`，Branch 选择 `main`，Folder 选择 `/root`。
5. 等 GitHub 生成网址。

## 添加到 iPhone 主屏幕
1. 用 iPhone Safari 打开 GitHub Pages 网址。
2. 点击分享按钮。
3. 点击“添加到主屏幕”。
4. 名称填“旅行者2.0”。
5. 点击添加。

## 功能
暗色游戏界面、RP/旅币/A7资产、夜潮印记、3天保底、浪花账冷却、强门禁、旅行驿站、Boss血条、地图/天气/混沌因子、日报结算、本地存档 localStorage。
